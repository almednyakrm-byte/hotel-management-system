# نظام إدارة فنادق (إدارة الحجوزات، الغرف، الخدمات)
==============================================

## نظرة عامة وغرض المشروع
نظام إدارة فنادق هو تطبيق ويب يهدف إلى تسهيل عمليات إدارة الفنادق، بما في ذلك إدارة الحجوزات، الغرف، والخدمات. يهدف المشروع إلى توفير حل متكامل ومرن يمكنه تلبية احتياجات الفنادق المختلفة.

## هيكل المشروع

## تشغيل البيئة
لتشغيل البيئة، اتبع الخطوات التالية:
1. تأكد من تثبيت Docker و Docker Compose على جهازك.
2. قم بفتح Terminal أو Command Prompt وانتقل إلى دليل المشروع.
3. قم بتشغيل الأمر `docker-compose up` لتشغيل الحاويات.
4. انتظر حتى يتم تشغيل الحاويات وفتح التطبيق في المتصفح.
